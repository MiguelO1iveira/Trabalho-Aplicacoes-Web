Exercício Operadores Bitwise no Java

Desenvolvi um exercício em Java por iniciativa própria para praticar operadores bitwise, simulando um sistema simples de permissões de usuário (leitura, escrita e execução). A ideia foi criar um exemplo prático para fixar o conceito de flags binárias e compreender como permissões podem ser controladas utilizando operações bitwise.

📌 Conceitos aplicados

  • Operador OR (|): para adicionar uma nova permissão sem alterar as existentes
  • Operador AND (&): para verificar se uma permissão específica está ativa
  • Operador NOT (~): utilizado em conjunto com o AND para remover uma permissão específica
  • do-while para gerenciar o loop do menu interativo
  • switch-case para organizar as opções do menu
  • if-else para validação das permissões

Trabalho de Funções Web (Faetec)

Desenvolvi um trabalho em grupo com o objetivo de apresentar a origem do basquete de forma organizada e visualmente atrativa, reunindo informações históricas sobre a criação do esporte, seu desenvolvimento ao longo do tempo e alguns elementos importantes relacionados às regras e competições.

A proposta foi criar uma página informativa para facilitar o entendimento sobre como o basquete surgiu, quem foi seu criador e como o esporte evoluiu até se tornar uma das modalidades mais populares do mundo.

📌 Conceitos aplicados

  • Organização de conteúdo: apresentação da história do basquete em uma estrutura clara e objetiva.
  • Navegação por menu: uso de links para outras áreas do site, como NBA, FIBA, NBB, Regras, Login e Registrar.
  • Conteúdo multimídia: inclusão de imagens para enriquecer a experiência visual da página.
  • Leitura informativa: explicação sobre a criação do basquete por James Naismith, o primeiro jogo e a evolução das regras do esporte.

Trabalho de Fundamentos de Desenvolvimento de Software

Desenvolvi este projeto com o objetivo de apresentar conceitos fundamentais da área de Engenharia de Software de forma estruturada e acessível, transformando conteúdos teóricos em uma interface web organizada e de fácil navegação.

A proposta foi consolidar conhecimentos adquiridos durante os estudos, utilizando uma página web como meio de apresentar temas importantes da área, como organização de sistemas, conceitos básicos de software e estruturação de conteúdo técnico.

📌 Conceitos aplicados

  • Estruturação com HTML: organização semântica do conteúdo em seções, facilitando a leitura e navegação do usuário
  • Estilização com CSS: aplicação de estilos para melhorar a apresentação visual e a experiência do usuário.
  • Organização de conteúdo técnico: transformação de conceitos teóricos em uma interface clara e compreensível.
  • Navegação entre seções: uso de links e estrutura de página para facilitar o acesso às diferentes partes do conteúdo.
